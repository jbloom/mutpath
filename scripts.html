ScriptsÂ¶

Installation of mutpath installs a set of scripts that can be run directly from the command line. These scripts perform all of the main functions of the package. You should be able to utilize the full functionality simply by running these scripts, without interacting directly with the Python code.

Here are the scripts:

  • mutpath_parse_to_beast_xml.py
  • mutpath_compact_trees.py
  • mutpath_get_paths.py
  • mutpath_annotate_tree.py
  • mutpath_make_digraph.py
